# Oklang - the language that is ok
- It's like C but it's **usable**.
- It's like C3 but it's **not ugly**.
- It's like Odin but it's **not ugly**.
- It's like Zig but it's **not crazed with explicit control flow**.
- It's like Rust but it's **not crazed with memory safety**.

That's all there is to it.
